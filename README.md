# sql-practice

This repository contains MYSQL and SQL practice queries.

## Topics Covered 
- Database Creation
- Table Creation
- Insert data
- Select
- From 
- Where clause
- order by
- Group by
- Aggregate Functions
- Having Clause
- Joins
       * Inner Join
       * Left Join
       * Right Join
       * Full Join
       * Self Join
       * Cross Join
- Indexing
- Transaction
- ACID properties
          * Atomicity (All or None)
          * Consistency (Correctness & rejects invalid data)
          * Isolation (works as only transaction even many transactions runs)
          * Durability (Crash safety)

## Tools used
- MySQL 8.0
- MySQL Workbench
- SQL Server Management Studio(SSMS)
